About this sample

This sample shows how to perform user input validation. If the entered value is not valid, a red border and a warning popup are shown.
The following criteria is checked:
- Name is required and should be at least 3 characters long.
- Register date is required, should be a valid date and should not be in the future.
- Tickets field is required and should be between 1 and 99.
- Winnings field is required and should be between 1000 and 10000.
