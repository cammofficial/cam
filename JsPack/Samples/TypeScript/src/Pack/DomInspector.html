About this sample

This sample shows how to build a tree dynamically in the Diagram and TreeView controls. The Diagram is arranged by the automatic tree layout algorithm. Tree nodes can be rearranged inside the TreeView control or moved to the ListView control below by using the built-in drag and drop capabilities of the TreeView and ListView controls. Tips are displayed by the Tooltip control.

This page requires a browser that supports HTML 5 Canvas element.
