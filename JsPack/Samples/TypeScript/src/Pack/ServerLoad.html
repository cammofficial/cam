About this sample

A sample server load web application. The chart shows number of users on a given connection between two stations in the network at each moment. The graphic includes data for the last 30 seconds. The diagram shows the servers and clients that build the network. By default the charts shows data only for the two most important connections, out of total 10. Users can select different connections and view their graphics. The gauge control provides data for the average count of users at any given moment.
Click on a link in the network to show its load. Click and drag with the mouse or hold 'Ctrl' to select several links. Click anywhere else on the diagram to return to default.
